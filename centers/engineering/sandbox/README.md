## 绝对隔离原则

```
沙盒内崩溃/死循环/异常  ≠  主环境污染
沙盒是独立宇宙，主环境是物理隔离的
```

### 物理隔离保障
1. **禁止从沙盒内部修改主环境配置**（crown/queue.json / crontab / SOUL.md）
2. **沙盒内脚本禁止调用 `rm -rf /` 类操作**
3. **沙盒日志默认不进入主日志流**
4. **沙盒最多使用 10% 系统配额**
5. **沙盒进程超时强制 kill**（`timeout` 封装）
